# agents-skills-for-japanese

このリポジトリは、Codex/Agents で「英語でのプロンプト入力」を支援・強制するためのスキル集です。

## ライセンス

MIT

## 目的

- 日本語を含む入力を英語に翻訳し、英語入力を促す
- 英語プロンプトの文法ミスやタイプミスを修正し、学習をサポートする

## 含まれるスキル

- `ja2en`: 日本語混じりの入力を英語へ翻訳し、英語で入力するように促す
- `english-grammer-checker`: 英語入力の文法・タイポを修正し、誤り箇所を説明する

## ディレクトリ構成

```text
.agents/skills/ja2en/SKILL.md
.agents/skills/english-grammer-checker/SKILL.md
```
